# e-Commerce Back-End

## Purpose
The back end for an e-commerce site. Utilizes an Express.js API and configures it to use Sequelize to interact with a MySQL database.

## Built With
* JavaScript
* Object-Relational Mapping (ORM)
* Node.js
* Express.js API
* MySQL2
* Sequelize
* dotenv
